## ðŸ”§ Troubleshooting

### "Your LaTeX won't compile on arXiv"

**Solution 1:** Check arXiv processing log
- Click "View LaTeX compilation log"
- Fix errors indicated
- Resubmit

**Solution 2:** Simplify LaTeX
- Remove custom packages
- Use only standard packages
- Convert complex figures to simple PDFs

**Solution 3:** PDF-only submission
- Upload pre-compiled PDF
- Include figures already in PDF
- Less flexible but guaranteed to work

### "Figures don't appear"

**Check:**
- Figure files in same directory as .tex
- Correct file extensions (`.pdf` not `.PDF`)
- Path in `\includegraphics` correct
- No spaces in filenames

### "Need endorsement"

**Options:**
1. Ask colleague with arXiv account
2. Request from paper authors you cited
3. Contact arXiv moderators (explain research)
